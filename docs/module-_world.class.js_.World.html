checkCollisions()

Checks for collisions between the sharky character and enemies in the level. This method sets up an interval that repeatedly checks if the sharky character is colliding with any enemies. If a collision is detected, it triggers the hurtSharky method on the sharky character and sets the isCurrentlyHurt flag.

setWorld()

Sets the world context for various game elements. This method assigns the current world instance to the `sharky` object, specific landscape elements, enemies in level 1, and status bars. - Assigns the world to `sharky`. - Assigns the world to landscape elements at indices 2, 3, 4, and 5. - Assigns the world to each enemy in `level1.enemies` if the enemy is an instance of `EndBoss`. - Assigns the world to each status bar in `statBars` if the status bar is an instance of `StatusBarLife`.
