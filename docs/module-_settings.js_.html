(inner) toggleFullscreen()

Toggles the fullscreen mode for the canvas element. This function checks the current fullscreen state and either requests fullscreen mode for the canvas element or exits fullscreen mode. It also updates the `isFullscreen` variable accordingly and calls `showHideControlScreen()` when entering fullscreen mode.
